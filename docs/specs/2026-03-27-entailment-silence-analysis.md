# Entailment 势函数分析：反向消息强度问题

> **Status**: Proposal
> **Date**: 2026-03-27
> **Scope**: theory/ entailment potential, inference_v2
> **Related**: belief-propagation.md §2.6, science-formalization.md §3.4

## 0. 摘要

belief-propagation.md §2.6 对 entailment 使用沉默势函数 ψ(A=0, B) = 1.0。在因子图的 C2（弱确认）检验中，silence 技术上满足 C2（P(A|B=1) > P_marginal(A)），但 **C2 效应极弱**（反向消息接近均匀），导致 science-formalization.md §3.4 的细因子图分解模式无法正常工作。

**实验结果**：严格按 theory 实现的细因子图上，V（伽利略结论）= 0.382、G（阻力假说）= 0.176，远低于文档 §3.5 "belief 趋向 1−ε" 的预期。

**根因**：ψ(A=0, B=1) = 1.0 > ψ(A=1, B=1) = p = 0.999，导致 factor→A 的反向消息比率 p/1.0 < 1，几乎不携带信息。

**修复**：将对称常数从 1.0 改为 ε = CROMWELL_EPS。C4 沉默性质不变，反向消息强度从 ~0.47 提升到 ~0.999，零新参数。

**是否有修改的必要**：取决于是否使用细因子图分解。粗因子图上差异极小（关键路径使用 induction/abduction），细因子图上原设计会导致分解模式失效。

## 1. 问题：Silence 的反向消息几乎无信息

### 1.1 反向消息推导

对单因子 G →[ent, p]→ B，设 B 的外部消息为 q = msg_B→f(1)（B 被观测支撑时 q ≈ 0.9）。

factor→G 消息：

| | G=1 | G=0 |
|--|-----|-----|
| silence (c=1.0) | pq + (1-p)(1-q) ≈ **q** | 1·q + 1·(1-q) = **1.0** |
| c = ε | 同上 ≈ **q** | ε·q + ε·(1-q) = **ε** |

归一化后 P(G=1|msg)：

| 模型 | P(G=1\|msg) | 比率 μ(G=1)/μ(G=0) |
|------|------------|---------------------|
| silence (c=1.0) | q/(q+1) = **0.474** | 0.9 : 1.0 ≈ **0.9** |
| c = ε = 0.001 | q/(q+ε) = **0.999** | 0.9 : 0.001 = **900** |

**Silence 的反向消息比率 < 1**：factor 在告诉 G "你稍微更可能为假"，即使 B 确实为真。C2 在因子图的边缘概率意义上技术满足（P(G|B=1) > P_marginal(G)，因为 marginal 已被 factor 拉低），但信号强度接近零。

### 1.2 原因

ψ(G=0, B=1) = **1.0** > ψ(G=1, B=1) = **p = 0.999**。在 silence 看来，"G 假且 B 真"比"G 真且 B 真"稍微更自然——因为 G=0 对 B 不约束（势=1.0），而 G=1 要求 B 以概率 p<1 为真（势<1.0）。

### 1.3 为什么 §2.6 表格的 C2 = ✓ 不算错但有误导性

§2.2 在贝叶斯网络模型下证明了 C2，那里前提变量没有父节点，边缘概率 = 先验。inference_v2 用的是因子图，factor 会把 A 的边缘概率从 π_A 拉低。用因子图正确的比较（P(A|B=1) vs P_marginal(A)），C2 技术上成立。但效应极弱（Δ ≈ +0.08），反向消息几乎无信息。

## 3. 修复方案

### 3.1 修改

将 ψ(premise=False, conclusion) 从 1.0 改为 ε = CROMWELL_EPS：

```
当前：  ψ(A=0, B=1) = 1.0,    ψ(A=0, B=0) = 1.0
修复后：ψ(A=0, B=1) = ε,      ψ(A=0, B=0) = ε
```

两个值相等（对称常数），所以 factor→B 消息在 A=0 时仍为 [ε, ε] → [0.5, 0.5]，**C4 沉默性质完全保持**。

### 3.2 四条弱三段论验证

| 三段论 | silence (c=1.0) | 修复后 (c=ε) |
|--------|----------------|-------------|
| C1（A真→B信升） | ✓ | ✓（不变） |
| C2（B真→A信升） | ✓（极弱，Δ≈0.08） | ✓（强，Δ≈0.10+） |
| C3（B假→A信降） | ✓ | ✓（不变） |
| C4 沉默（A假→B不受影响） | ✓ | ✓（ψ(0,1)=ψ(0,0)=ε，对称） |

### 3.3 代码改动

`potentials.py` 第 103-104 行：

```python
# 当前
if not all(assignment[v] == 1 for v in premises):
    return 1.0

# 修复后
if not all(assignment[v] == 1 for v in premises):
    return CROMWELL_EPS ** len(conclusions)
```

`exact.py` 第 57-62 行对应修改：将 `1.0` 替换为 `eps`。

共 2 处改动。零新参数（复用已有的 CROMWELL_EPS）。

## 4. 是否有修改的必要

| 使用场景 | 是否需要改 | 理由 |
|---------|-----------|------|
| **只用粗因子图** | **不急** | 关键路径走 induction/abduction（已是 noisy-AND），entailment 只用于内部逻辑推导，silence 影响极小 |
| **用细因子图分解（§3.4/§3.5）** | **必须改** | 分解模式将所有 weakpoint 替换为 entailment + equivalence，反向消息成为唯一的假说支撑通路，silence 使此通路失效（V=0.38 vs 预期 0.99） |
