# Gaia Language v3 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Simplify Gaia Language v2 to v3 — remove strategy tactics, streamline proof state to hole detection, redesign `#claim_relation` with auto-premise.

**Architecture:** v3 is a refactor of v2 on branch `feat/typst-gaia-lang-poc`. Changes span the Typst library (remove 9 tactics, simplify `#claim_relation`), Python loader/renderer/proof_state (remove proof trace handling), and test fixtures (rewrite with prose instead of tactics). v1 code (`knowledge.typ`, `chain.typ`, `lib.typ`) is untouched.

**Tech Stack:** Typst (DSL), Python 3.12+, pytest, ruff

**Spec:** `docs/superpowers/specs/2026-03-18-gaia-lang-v3-design.md`

**Base branch:** `feat/typst-gaia-lang-poc` (PR #149)

- [ ] **Step 2: Rewrite `#claim_relation` — remove proof block, auto-populate premises**

Replace the `claim_relation` function. Key changes:
- Remove proof block support (no `_proof_active`/`_proof_conclusion` toggling)
- Auto-push `between` items to `_gaia_proof_premises`
- Optional description body (single content block, not two)

```typst
// ── claim_relation: structural constraint between declarations ──
// between parameters automatically become premises.
// No proof block needed — the relation declaration is self-complete.
#let claim_relation(name, type: "contradiction", between: (), ..args) = {
  let positional = args.pos()
  let description = if positional.len() > 0 { positional.at(0) } else { none }

  _register_node(name, type, if description != none { description } else {
    [#type: #between.join(", ")]
  })

  // Auto-populate premises from between
  for b in between {
    _gaia_proof_premises.update(p => { p.push((name, b)); p })
  }

  // Emit constraint
  _gaia_constraints.update(constraints => {
    constraints.push((
      name: name,
      type: type,
      between: between,
    ))
    constraints
  })

  if description != none {
    [#figure(kind: "gaia", supplement: none, outlined: false)[
      *#name.replace("_", " ")* (#type): #description \
      _Between: #between.join(", ")_
    ] #label(name.replace("_", "-"))]
  } else {
    [#figure(kind: "gaia", supplement: none, outlined: false)[
      *#name.replace("_", " ")* (#type) \
      _Between: #between.join(", ")_
    ] #label(name.replace("_", "-"))]
  }
}
```
