Key Interactions

🔍 Click a node

Highlights entire upstream reasoning chain — all premises, strategy nodes, and their premises, recursively. Dimmed nodes stay visible but faded. Detail panel slides in from right showing content, prior→belief, and reasoning edges.

🔗 Cross-module trace

When a premise comes from another module, it shows as a dashed "external" node with a clickable link. Click it → navigate to that module's subgraph, auto-highlighting the referenced node.

📊 Belief badges

Each node shows belief value as color-coded badge. Green (≥0.7), amber (0.4-0.7), red (<0.4). Prior→belief delta shown on hover.

🗺️ Mini-map

In module subgraph view, a small overview in the corner shows current viewport position and which nodes are highlighted, for orientation in large modules.
