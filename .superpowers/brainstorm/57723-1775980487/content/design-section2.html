ELK Layout Engine

Why ELK over dagre/cola: ELK (Eclipse Layout Kernel) is purpose-built for hierarchical/layered graphs. It natively supports:

  • Compound nodes — modules as containers with internal layout
  • Port-based routing — edges connect to specific sides of nodes, reducing crossings
  • Layered algorithm — automatically assigns nodes to layers (settings at top, conclusions at bottom)
  • 200+ nodes — ELK handles industrial-scale graphs (Eclipse IDE uses it)

Integration: Use elkjs (WASM build, ~180KB gzipped) with React Flow or Cytoscape. Layout computed in web worker to avoid blocking UI.
